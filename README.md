# Playwright POM Automation Project

## Overview
This project contains automated end-to-end test scripts using Playwright with Page Object Model (POM).

## Tasks Covered
- Customer Registration
- Login Functionality
- Update Profile

## Technologies Used
- Playwright
- JavaScript
- POM

## How to Run

npm install
npx playwright test

## Project Structure

pages/
LoginPage.js
RegisterPage.js
AccountPage.js

tests/
login.spec.js
register.spec.js
updateProfile.spec.js

## Test Report

npx playwright show-report
